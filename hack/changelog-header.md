### Linux

```shell
curl -L https://github.com/jenkins-x-plugins/jx-preview/releases/download/v{{.Version}}/jx-preview-linux-amd64.tar.gz | tar xzv 
sudo mv jx-preview /usr/local/bin
```

### macOS

```shell
curl -L  https://github.com/jenkins-x-plugins/jx-preview/releases/download/v{{.Version}}/jx-preview-darwin-amd64.tar.gz | tar xzv
sudo mv jx-preview /usr/local/bin
```
